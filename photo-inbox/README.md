# 📥 Photo Inbox

Drop any photos you want on the website into this folder. This is a staging
area — nothing here is shown on the site until it gets wired in.

**How to use it:**

1. Copy photos in with **descriptive filenames**, e.g.
   `travel-tokyo-shibuya-night.jpg`, `hero-alternate.jpg`, `story-graduation.jpg`.
   The filename is how I'll know where a photo belongs.
2. Optional: group them in subfolders (`travel/`, `story/`, `home/`, …) —
   same idea, the folder name hints at the destination.
3. Tell Claude "implement the photos in the inbox" (or which specific ones).
   They'll be optimized, moved into `public/photos/`, and placed into the
   right pages. Dark, moody, night-time shots fit the current theme best.

Anything still in here is treated as *not yet placed*.

Note: this folder is gitignored (originals are heavy) — only the
web-optimized copies in `public/photos/` are committed.
